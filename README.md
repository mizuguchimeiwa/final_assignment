# CIFAR-10 CNN Improved Version
This repository contains an improved version of the PyTorch official CIFAR-10 image classification tutorial.

The original tutorial code was refactored and improved by applying concepts
learned in class, such as environment management with uv, type hints, pytest-based testing, and code modularization.
